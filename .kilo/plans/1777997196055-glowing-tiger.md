## 5. Algoritmo de Ordenamiento

### Dentro de cada grupo:
1. **Primario:** `apellido` (A-Z), luego `nombre` (A-Z)
2. **Secundario (opcional):** `plan.nombre` para desempate

### Cálculo de estado de membresía:
```typescript
function getMembershipStatus(socio: SocioConEstado): 'vigente' | 'porVencer' | 'vencida' {
  const fechaVencimiento = addDays(new Date(socio.fechaInicio), socio.plan.duracionDias)
  const diasRestantes = differenceInDays(fechaVencimiento, new Date())
  
  if (diasRestantes < 0) return 'vencida'
  if (diasRestantes <= 7) return 'porVencer'
  return 'vigente'
}
```

## 6. Implementación de Animaciones (framer-motion)
